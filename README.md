### `keys`

You must add a directory `./keys/` containing the following keys:
- `AWS_ACCESS_KEY.txt`
- `AWS_SECRET_KEY.txt`
- `cache_key.txt`
- `log_AWS_ACCESS_KEY.txt`
- `log_AWS_SECRET_KEY.txt`
- `secret_key.txt`
